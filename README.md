<!-- GETTING STARTED -->
## Getting Started

### Prerequisites
- Debian Minimal Installation
- Internet Connection
### Installation
``
  mkdir ~/.apps
  git clone https://codeberg.org/DarkestMuon/Config ~/.apps/Config
  chmod +x ~/.apps/Config/scripts/install.sh
  ~/.apps/Config/scripts/./install.sh
``
